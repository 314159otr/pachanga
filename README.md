# Pachanga
This is a toy project for tracking the football matches that we play every week (one can wish).
It's an Android app with only 2 screens:

## Players Stats Screen
<p align="center">
  <img alt="Screenshot of Players Stats screen" src="https://github.com/user-attachments/assets/7cb6663d-9903-42d5-86e1-824dd3b20b30" width="300">
</p>
Here we have a table with sortable columns and a season selector.
There is also a floating download button to download new versions of the database that I update after every match. The button disappears when you scroll through the table to allow better visibility.

## Matches Screen
<p align="center">
  <img alt="Screenshot of Matches screen" src="https://github.com/user-attachments/assets/55679525-27de-4663-adc1-566333096cbb" width="300">
</p>
Here we have a football field with the players and stats for a given match.
There is a match selector with every match played. When the current match is pressed, a calendar pop-up appears, allowing us to select any match easily.
